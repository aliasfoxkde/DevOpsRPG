# DevOpsQuest 🎮

**An Open Source Gamified DevOps Learning Experience**

[![Live Game](https://img.shields.io/badge/Play-DevOpsQuest-amber?style=for-the-badge&logo=gamepad)](https://f7b4e42f.devopsquest.pages.dev)
[![GitHub](https://img.shields.io/badge/GitHub-Repository-333?style=for-the-badge&logo=github)](https://github.com/aliasfoxkde/DevOpsRPG)

---

**Live Game**: https://f7b4e42f.devopsquest.pages.dev

DevOpsQuest is a gamified learning platform that makes mastering DevOps concepts fun and engaging. Level up your character by completing real-world challenges across Linux, Git, Docker, Kubernetes, AWS, CI/CD, and more!
